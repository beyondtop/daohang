# 附录1 核心素养学段特征

核心素养在各学段的表现特征如表9至表12所示。

<center><b>表9 信息意识学段特征</b></center><br/>

|  学段  |  特征  |
|  ----  |  ----  |
|  第一学段<br/>（1～2年级）  |  <p>1.在教师指导下，体验使用数字设备获取日常学习与生活中所需要的信息。</p><p>2.具有学习与使用数字设备的兴趣与意愿，养成数字设备使用的好习惯。</p><p>3.知道信息有真伪之分，增强保护个人隐私的意识。</p><p>4.乐于使用数字设备与同伴交流、分享信息。</p>  |
|  第二学段<br/>（3～4年级）  |  <p>1.了解获取信息的多种途径，并学会使用这些途径获取信息。</p><p>2.能通过典型的应用实例了解编码在信息社会中的重要价值。</p><p>3.具有寻找有效数字平台、资源的意愿，乐于参与在线方式下的协同学习与活动，体验信息科技在生活中的用途。</p>  |
|  第三学段<br/>（5～6年级）  |  <p>1.初步认识信息处理的作用与价值。</p><p>2.在完成任务过程中，有意识地寻求恰当的算法解决问题，了解算法的优势和局限。</p><p>3.按照任务需求，有意识地应用反馈优化解决问题的过程。</p><p>4.理解科学精神、原创精神，具有将创新理念融入自身学习、生活中的意识。</p>  |
|  第四学段<br/>（7～8年级）  |  <p>1.认识到互联网、物联网和人工智能对社会发展的影响，善于使用信息科技解决学习和生活中的问题。</p><p>2.能根据学习或生活需求，分析数据需求，评估数据来源，有效辨别数据的可靠性和时效性。</p><p>3.具有较强的数据安全意识，能预防个人及他人隐私的泄露。</p><p>4.崇尚科学精神、原创精神，具有自主动手解决问题、掌握核心技术的意识。</p>  |

<center><b>表10 计算思维学段特征</b></center><br/>

|  学段  |  特征  |
|  ----  |  ----  |
|  第一学段<br/>（1～2年级）  |  <p>1.喜欢发现问题，乐于用图符的方式进行表达。</p><p>2.能说出使用常见数字设备的基本步骤。</p><p>3.合理使用数字设备，对数字设备与真实情境的关联感到好奇。</p>  |
|  第二学段<br/>（3～4年级）  |  <p>1.感受身边无处不在的数据。</p><p>2.了解信息的呈现和传递方式，合理使用字符编码表示信息。</p><p>3.能将问题进行分解，并用文字或图示描述解决问题的顺序。</p>  |
|  第三学段<br/>（5～6年级）  |  <p>1.通过体验身边的算法，了解算法的特征和效率，会用自然语言、第三学段流程图等方式描述算法。</p><p>2.在一定的活动情境中，能对简单问题进行抽象、分解、建模，制订简单的解决方案。</p><p>3.验证解决方案，反思问题解决的过程和方法，并对其进行优化。</p>  |
|  第四学段<br/>（7～8年级）  |  <p>1.能在真实情境中发现问题，提取问题的基本特征，对问题进行抽象、分解、建模，制订解决方案。</p><p>2.能执行问题解决方案，通过对数据的采集与分析，反思和优化解决方案，并将其迁移运用于解决其他问题。</p>  |

<center><b>表11 数字化学习与创新学段特征</b></center><br/>

|  学段  |  特征  |
|  ----  |  ----  |
|  第一学段<br/>（1～2年级）  |  <p>1.在教师指导下，选择数字资源开展学习活动。</p><p>2.辨认基本的数字设备、平台和资源，运用给定的数字设备收集学习资源。</p>  |
|  第二学段<br/>（3～4年级）  |  <p>1.了解数字化学习环境的优势，能在数字化学习环境中开展学习活动。</p><p>2.在教师指导下，运用在线工具开展合作学习与创新活动，创作简单作品。</p><p>3.使用数字设备组织并呈现收集的数据，借助思维导图等方式表示数据之间的关系，支撑自己的观点。</p>  |
|  第三学段<br/>（5～6年级）  |  <p>1.在问题情境中，能够利用信息科技开展数字化学习与交流，合作解决学习问题。</p><p>2.根据学习任务，合理选择数字设备、平台和资源，通过任务分解的方式提高学习效率。</p><p>3.按照任务需求，运用信息科技获取、加工、管理学习资源，开展数字化探究和创新活动。</p>  |
|  第四学段<br/>（7～8年级）  |  <p>1.合理运用信息科技获取、加工、管理和评价学习资源，解决学习问题。</p><p>2.能依托网络平台、智能工具开展远程学习、小组交流、协同创新等活动。</p><p>3.按照任务需求，能选择合适的数字设备、平台和资源，合作完成数字化创新活动的创意、规划与实施。</p><p>4.认识到原始创新对国家可持续发展的重要性。</p>  |

<center><b>表12 信息社会责任学段特征</b></center><br/>

|  学段  |  特征  |
|  ----  |  ----  |
|  第一学段<br/>（1～2年级）  |  <p>1.能认识信息科技给生活带来的便利，健康使用数字设备。</p><p>2.能在教师指导下安全文明地使用互联网，并遵守网络道德规范。</p><p>3.在各种在线活动中，能认识到个人信息保护的重要性，尊重数字作品所有者权益。</p>  |
|  第二学段<br/>（3～4年级）  |  <p>1.知道按照规范开展信息活动，认识数字身份的唯一性与信用价值。</p><p>2.知道常用信息保护方法，主动保护个人信息。</p><p>3.了解威胁数据安全的因素，有意识地保护数据，遵守相关法律法规。</p>  |
|  第三学段<br/>（5～6年级）  |  <p>1.了解知识产权的概念，认识知识产权保护和应用的重要意义。</p><p>2.在信息分享与交流活动中，能通过合理的方法辨别与抵制不良信息,尊重他人观点,理性发表个人观点。</p><p>3.初步具备数据安全防护能力，能保护个人及他人隐私，自觉维护网络安全和数据安全。</p>  |
|  第四学段<br/>（7～8年级）  |  <p>1.具有良好的协作意识与习惯，乐于帮助他人开展信息活动，负责任地共享信息和资源，并尊重他人知识产权。</p><p>2.了解网络虚拟身份的作用，保护个人和他人隐私，认识网络法律法规和伦理道德规范对信息社会有序发展的保障作用。</p><p>3.能正确认识人工智能对社会的影响以及对伦理与安全的挑战，合理使用人工智能。</p><p>4.能遵循信息科技领域的伦理道德规范，明确科技活动中应遵守的价值观念、道德责任和行为准则。</p><p>5.能认识自主可控技术的重要意义，树立科技自立自强的意识。</p>  |
